# 🔍 SearchKaro - Full-Stack Directory Platform

**SearchKaro** ek high-performance directory application hai jo Nagpur ke business aur locations ko search karne ke liye banaya gaya hai.

## 🚀 Live Links
- **Frontend (Vercel):** [https://searchkaro-app-three.vercel.app](https://searchkaro-app-three.vercel.app)
- **Backend (Render):** [https://searchkaro-app.onrender.com](https://searchkaro-app.onrender.com)

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT (JSON Web Tokens) & Bcrypt

## ⚙️ Features
- **Secure Auth:** Signup aur Login functionality with JWT.
- **Search Engine:** Regex-based searching for categories and locations.
- **Dashboard:** Dynamic statistics for ratings and reports.
- **Responsive:** Mobile aur Desktop dono ke liye optimized.

## 📁 Project Structure
- `/frontend`: Vite + React UI components.
- `/backend`: Express API routes and Mongoose models.
